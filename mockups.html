Website Mock-ups

All mock-ups were created with Photoshop. All of them have a width of 720 pixels and vary on height. This mock-ups were created to visualize a food website, more specically breakfast bowls. It's intended for the website to display different breakfast recipes according to different occasions and days of the week. The website is also going to have an "About" section and a "Contact" section as you can see in the mock-ups, which have a coherent theme throghout. To place each element of the mock-ups in the right place I used guide layouts that differed from each webpage.
